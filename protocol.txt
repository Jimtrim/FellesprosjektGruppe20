Calendar application server <-> client protocol

Feel free to change!

Status codes:
		The norm: xx0 = SUCCESS, xx9 == ERROR
		
		100 Login SUCCESS
		109 Login ERROR
		190 Logout SUCCESS
		199 Logout ERROR
		
		200 Appointment CREATE SUCCESS
		209 Appointment CREATE ERROR
		210 Appointment UPDATE SUCCESS
		219 Appointment UPDATE ERROR
		290 Appointment DELETE SUCCESS
		299 Appointment DELETE ERROR
		
		300 Room CREATE SUCCESS
		309 Room CREATE ERROR
		310 Room RESERVE SUCCESS
		319 Room RESERVE ERROR
		320 Room UNRESERVE SUCCESS
		321 Room UNRESERVE ERROR
		390 Room DELETE SUCCESS
		399 Room DELETE ERROR
		
		910 General NEW UPDATES
		911 General NO UPDATES
		919 General REQUEST ERROR
	 
-- 1xx : User --
Login:
	Client request:
		LOGIN <String: username> <String: password>
	Server reply:
		109 <Explanation>
		101 <Int: userID> <String: firstName> <String: lastName>

Logout:
	Client request:
		LOGOUT
	Server reply:
		190


-- 2xx : Appointment --
Create Appointment:
	Client request:
		APPT CREATE <String: name> <String: description> <Long: startTime> 
		   <Int: roomID> <Int: Duration> [ <String: participant> ... ]
	Server reply:
		200 <Int: id> <String: name>
		209 <Explanation> 

Update Appointment:
	Client request:
		APPT UPDATE <Int: id> <String:name> <String:description> <Long:startTime> <Int:roomID> <Int:Duration>
		   [ <String: participant> ...]
	Server reply:
		210 <Int: id> <String: name>
		219 <Explanation>

Delete Appointment:
	Client request:
		APPT DEL <Int: id>
	Server reply:
		290 <Int: id> <String: name>
		299 <Explanation>

Appointment information from server:
	Server sends:
		APPT <id> <name> <description> <startTime> <duration> [<participant> ...]
	Client responds:
		220 <id>

-- 3xx : Room --
Create Room:
	Client request:
		ROOM CREATE <String: name> <String: descrition> <Int: Capacity> 
	Server reply:
		300 <Int: id> <String: name>
		309 <Explanation>

Reserve Room:
	Client request:
		ROOM RES <Int: id>
	Server reply:
		310 <Int: id> <String: name>
		319 <Explanation>

Unreserve Room:
	Client request:
		ROOM UNRES <Int: id>
	Server reply:
		320 <Int: id> <String: name>
		329 <Explanation>

Delete Room:
	Client request:
		ROOM DEL <Int: id>
	Server reply:
		290 <Int: id> <String: name>
		299 <Explanation>

-- 9xx : General --
General check updates:
	Client request:
		UPDATE <Int: id>
	Server reply:
		910 <Int: number of updates>
		911 <Ignored>                 Sent when no updates are available.
	
For any unspecific error, the server sends:
		919 <Explanation>
The client should probably retry the request.
